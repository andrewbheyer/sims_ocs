============
Installation
============

Installation of the Operations Simulator can be done via a :ref:`source installation<source-installation>` for Linux systems and via :ref:`Docker installation<docker-installation>` for macOS systems. The Docker route is **NOT** recommended for Linux due to bind volume permission issues.
